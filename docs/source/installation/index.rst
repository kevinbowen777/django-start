Installation
============

This project has been developed and tested entirely on a Debian Linux system.  This project was built using poetry_ as its Python dependency management tools.  The instructions provided within this documentation reflect that assumption. It is intended to be run either locally or within a Docker container.

Cloning the project
-------------------

To install the ``django-start`` project, run the following command in your terminal:

.. code-block:: console

   $ git clone https://github.com/kevinbowen777/django-start.git
   $ cd django-start

Depending on your needs and preferences, see either the
:doc:`local_installation` or :doc:`docker_installation` page for additional
installation instructions.
